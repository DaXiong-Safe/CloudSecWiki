## Local Deployment
1. Cloning project to local

```bash
git clone https://github.com/selefra/CloudSecWiki.git
cd CloudSecWiki
```

2. Install the `npm` dependency package and build

```bash
npm install
npm run build
```

3. Copy the files in the `build` directory to the nginx service web directory
